Terminal Status

A VeriStand channel updated by this custom device that indicates the status of the parent terminal. This table details the possible terminal status channel values for the bus controller and remote terminals:

Bus Controller Remote Terminal
0: Running 0: Running (Simulated)
1: Stopped 1: Running (Monitor)
2: Paused 2: Stopped
3: Not Configured 3: Paused
4: Not Configured

You can perform the following actions with this channel:

  • View on a screen.
  • Assign an alias.
  • Log values.
  • Map to other channels.
  • Connect to other custom devices.

This channel has the following configuration settings:

  • Name—Specifies the name of the channel in the system definition.
  • Description—Describes the status channel.
