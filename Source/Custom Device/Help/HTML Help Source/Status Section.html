Status

Contains error counters and other status indicators for the MIL-STD-1553 custom device.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.

This section contains the following channels which count the receive (Rx) errors detected in hardware and returned in the status word:

  • General Protocol Error Count—Count of words where a Protocol Error detected.
  • Improper Number of Words Error Count—Count of errors that occurred because the number of detected words differed from the number of configured words for a message.
  • Manchester Error Count—Count of Manchester encoding errors. Manchester encoding requires that each bit has at least one transition and occupies the same time.
  • Monitor Error Count—Count of words in which any of the following errors occurred: General Protocol Error, Wrong Sync Error, Improper Number of Data Words Error, Manchester Error, Parity Error, or Word Error.
  • Parity Error Count—Count of words where a Parity Error detected.
  • Unconfigured Frames Error Count—Count of frames received that were not configured in the hardware configuration file.
  • Word Error Count—Count of messages where a Word Error detected.
  • Wrong Sync Error Count—Count of words where the Wrong Sync Error detected.
