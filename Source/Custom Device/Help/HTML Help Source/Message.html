Message

Identifies each MIL-STD-1553 message payload. Each raw message is composed of a number of unsigned, 16-bit (U16) words. This custom device further supports the encoding of scaled parameters within a raw word or spanning multiple words.

Message Type Number of Words
Data Message 1..31 U16 words encoding 1..N parameters
Mode Code

Mode Codes 0..15 have zero associated data words.

Mode Codes 16..31 each have one associated data word.

This section has the following configuration settings:

  • Name—Specifies the name of the section in the system definition.
  • Description—Describes the section.
